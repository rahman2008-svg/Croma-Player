Build Instructions

Clone the repository:

git clone https://github.com/rahman2008-svg/Croma-Player.git
cd Croma-Player

Build Debug APK:

chmod +x gradlew
./gradlew assembleDebug

Build Release APK:

./gradlew assembleRelease
